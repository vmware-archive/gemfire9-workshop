2.2  Examine how the sample Authorization class matches privileges to Principals.

public void init(Principal principal,

                   DistributedMember remoteMember,

                   Cache cache) throws NotAuthorizedException {

 

    if (principal != null) {

      String name = principal.getName().toLowerCase();

      if (name != null) {

      this.name = name;

        if (name.equals("root") || name.equals("admin") || name.equals("administrator")) {

          addReaderOps();

          addWriterOps();

          this.allowedOps.add(OperationCode.REGION_CREATE);

          this.allowedOps.add(OperationCode.REGION_DESTROY);

        }

        else if (name.startsWith("writer")) {

          addWriterOps();

        }

        else if (name.startsWith("reader")) {

          addReaderOps();

        }

      }

}

 In the init method you assign privileges to Principals.  This is a sample but can also be loaded via XML, a file, a central server, or other.

Note how writer principals cannot read and how reader principals cannot write.

 

2.3  Examine how the sample authorization class authorizes operations.

public boolean authorizeOperation(String regionName, OperationContext context) {

 

    OperationCode opCode = context.getOperationCode();

 

    boolean isAuthorized = this.allowedOps.contains(opCode);

   

    if (!isAuthorized)

    return isAuthorized;

 

    // this code extends the generic check above and forbids writer0 to write key2

    if (opCode == OperationCode.PUT && name !=null && name.equalsIgnoreCase("writer0")) {

    PutOperationContext putOpContext = (PutOperationContext) context;

    if (putOpContext.getKey().toString().equalsIgnoreCase("key2"))

          isAuthorized = false;

    }

   

     return isAuthorized;

  }

The authorizeOperation method matches the current operation to the allowed operations for the Principal executing the operation. 

Note how in this example writer0 principals cannot write “key2”. Again, this is a sample and can be extended for your security protocol.
